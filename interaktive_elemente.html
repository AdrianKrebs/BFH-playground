Interaktive Elemente

details

Das <details> tag ermöglicht die Anzeige von zusätzlichen Informationen welche der User anzeigen und wieder verstecken kann.

Attribute

Nur globale Attribute

Beispiel

Copyright 1999-2014.

- by Refsnes Data. All Rights Reserved.

All content and graphics on this web site are the property of the company Refsnes Data.

summary

Das <summary> Tag definiert den sichtbaren Header <details> Element. Dieser Header ist klickbar und zeigt/versteckt die dazugehörigen Details

Attribute

Nur globale Attribute

menu

ACHTUNG: Ist nur von Firefox unterstützt.
Das <menu> Tag definiert eine Liste/Menu von Befehlen Dieses Tag wird für Context Menus, Toolbars und fürs Auflisten von Controls und Commands verwendent. Kann das bestehende Rechtsklick-Context-Menu im Firefox erweitern.

Attribute

  • type: Art des Menu

menuitem

ACHTUNG: Ist nur von Firefox unterstützt.
Das <menuitem> Tag definiert ein menu item das der User aufrufen kann.

Attribute

  • checked: Soll menu-item angewählt sein beim laden der Seite
  • icon: Icon des command items
  • label: Name des menu-items
  • type: Art des items. Checkbox, command, radio
  • default Markiert Command als default

Beispiel

Right-click inside this box to see the context menu!

This example currently only works in Firefox!
